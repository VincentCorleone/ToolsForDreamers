每种动物都以它曾经所是的留下痕迹（译者注：留下躯体的残骸）；唯独人类以他所创造的留下痕迹。 - 杰。布鲁诺威士忌

看看你的四周，很有可能大部分你看到的会是人类创造力的产物：书本、大楼、电脑、家具、道路、电视、电话、灯泡、飞机和音乐…… 这个名单可以永远地延续下去。所有的这些东西在某个时刻只是某人脑袋里的一个梦，而令人感到神奇的是，这些梦在今天都进入了现实。是什么使得这些梦成为可能？它是如何发生的？这本书围绕着将梦想转变为现实的一些工具展开探讨。
